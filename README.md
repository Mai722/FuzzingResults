# FuzzingResults


This repository contains the results obtained by running seven fuzzers: AFLfast, AFLgo, AFLplusplus, Fairfuzz, LearnAFL, MoPT, and Superion. 


It is dividen into four main parts:

## FuzzingRawResults
Here are the raw results of fuzzing, i.e. the unprocessed results of all the fuzzers. This directory is divided first by fuzzer and within each fuzzer there is a folder for each of the SUTs that have been tested.
